1. IP 주소란 무엇인가?

IP 주소는 인터넷에 연결된 기기를 식별하는 고유한 집 주소입니다.

핵심 역할은 수많은 기기 중에서 서로를 구분하고, 데이터가 정확한 목적지로 찾아가도록 안내합니다.

IP의 구분 : 공인 IP와 사설 IP

  • 공인 IP (Public): 전 세계에서 유일한 주소로, 인터넷에서 우리 집(또는 회사)을 나타냅니다.
  • 사설 IP (Private): 집이나 회사 내부(공유기 안)에서만 사용하는 내부용 주소입니다.

2. Live Server 주소의 의미

이 주소는 개발자가 자신의 컴퓨터를 실제 웹 서버처럼 동작시켜, 코드 변경 시 웹페이지를 바로 확인하기 위해 사용됩니다.

Live Server”는 실제 서버처럼 동작하지만, 개발자의 컴퓨터 안에서만 임시로 작동하는 도구이다.

실제 현실에서 모델하우스와 비슷한 점이 있는데, 진짜 집이 아니라 판매용으로 만들어둔 집이 이지만, 실제처럼 들어가보고 체험할 수 있습니다.

왜 Live Server를 사용하는가?

  1. Live Server는 http:// 프로토콜을 사용하는 임시 로컬 서버를 열어 실제 웹사이트처럼 작동시킵니다.
  2. 사용HTML 파일을 직접 열면(file:///) 실제 서버 환경과 다르게 작동하거나 자동 새로고침이 되지 않습니다.

주소 구성 해설

  • 127.0.0.1 : 내 컴퓨터 자신 (localhost)
  • :5500 : Live Server가 사용하는 포트 번호(방 번호)
  • /경로/파일.html : 서버에 요청하는 파일의 위치
